**Button** — the primary tactile control; use for any action. Springy molded-plastic press, pill shape.

```jsx
<Button variant="primary" size="md">Catch</Button>
```

Variants: `primary` (red), `secondary` (blue lens), `neutral` (white outlined), `ghost`. Sizes `sm|md|lg`. Props: `block`, `disabled`, `iconLeft`, `iconRight`.
